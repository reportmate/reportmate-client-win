# Speedtest CLI

Place the Ookla Speedtest CLI binary (`speedtest.exe`) in this directory.

Download from: https://www.speedtest.net/apps/cli

The binary is bundled into the MSI/NUPKG/ZIP installer and deployed to:
`C:\Program Files\ReportMate\speedtest\speedtest.exe`

Used by the Network module to collect download/upload speed and latency metrics.
